Function Exclusion Monitor

Preface: This is a cognitive functions test that (which I assume) has a different scoring algorithm than usual. Each function starts at a score of 0 and each question if ticked deducts a point to that given function. That is, instead of trying to determine what you are, it tries to use process of elimination to determine what you are not (which some find to be more accurate). This also means the questions are attuned to which function you are not, as opposed to what you are. Of course, you should try and not let this affect/bias your decisions, and it probably would've been better if you didn't read this. Anyhow, the source for the questions can be found here, which seems to be moderately acclaimed. As for ethics, this entire website is vibecoded, all I did was add the questions.

Instructions: Read the statements below. Check the box if you GENERALLY relate to the statement.

Note: The questions are randomized. Try not to overthink which "type/function" the question belongs to.
